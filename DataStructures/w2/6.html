Pointer and Pointee

每個指標(pointer)一定指向一個實體的記憶體空間，稱為它的pointee。有了pointee的pointer才可以做dereference，否則對一個沒有指向實體空間的pointer作dereference，會造成當機。

#include <stdio.h>

int main(){

    int count = 10;
    int *p; // p is a pointer
    int *p2;

    p = &count; // & is used to get the address of the count, count is pointee
    p2 = p;

    printf("The count is %d\n", count); // 10

    *p = 20;
    printf("The count is %d\n", count); // 20

    *p2 = 30;
    printf("The count is %d\n", count); // 30

    return 0;
}
