Function重點

寫function的第一個步驟先思考function的「輸出」與「輸入」，「輸入」指的是傳給function的參數，或稱argument，他們的個數與型態。「輸出」則為function要return的value與其型態。根據function的傳入參數與傳回值，可以將function分成下列四種型態。

  1. 有傳入參數，有傳回值
  2. 有傳入參數，沒有傳回值
  3. 沒有傳入參數，沒有傳回值
  4. 沒有傳入參數，有傳回值

以下是將原始程式中的計算sum與max改寫成2個function的範例。這兩個function都是屬於第一種型態的function，有傳入參數，有傳回值。

#include <stdio.h>
#include <stdlib.h>

// 宣告Function原型
int sum(int, int, int);
int maximum(int, int, int);

// 定義Function內容
int sum(int a, int b, int c) {
    return a + b + c;
}

int maximum(int a, int b, int c) {
    int max = a;
    if (b > max)
        max = b;
    if (c > max)
        max = c;
    return max;
}

int main() {
    int a, b, c;
    printf("Input three different integers:");
    scanf("%d%d%d", &a, &b, &c);
    printf("Sum is %d\n", sum(a, b, c));
    printf("Max is %d\n", maximum(a, b, c));
    return 0;
}
