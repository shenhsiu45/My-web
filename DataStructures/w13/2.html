良心題: 求和

以下是您提供的 C 語言程式碼:

#include <stdio.h>
int sum(int);
int main(){
    int n;
    
    while(scanf("%d", &n)!=EOF){
        printf("%d\n", sum(n));
    }
    
    return 0;
} 
int sum(int n){
    if(n==0||n==1)
        return n;
    else return n+sum(n-1);
}

該程式接受從標準輸入中輸入的整數，並計算從1到該整數的總和。

請在合適的編譯環境中執行該程式。
